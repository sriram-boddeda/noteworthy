# **App Name**: Noteworthy

## Core Features:

- Rich Text Editor: Rich Text Note Editor: A professional-grade editor with comprehensive formatting options (font, size, headings, etc.).
- Markdown/LaTeX Support: Markdown/LaTeX Editor: Support for standard Markdown and LaTeX syntax with live preview and export functionalities. Formats to be supported: PDF, DOCX and HTML.
- Calculator Notes: Calculator Note with Live Computation: Allows users to define variables and perform calculations within notes, displaying results in real-time, acting as a calculation tool.
- Export: Export to different formats. PDF, DOCX, HTML.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of professionalism and focus.
- Background color: Light Gray (#F5F5F5) for a clean and distraction-free writing environment.
- Accent color: Teal (#009688) for interactive elements and highlights to provide a subtle contrast.
- Headline font: 'Poppins', a geometric sans-serif, to provide a clean and modern appearance. Body Font: 'PT Sans', to ensure great readability for large passages of text
- Minimalist icons that visually represent formatting options and note types. Follow Material Design guidelines.
- Clean, distraction-free layout that adapts to different screen sizes.
- Subtle transitions and animations to provide a polished user experience.
